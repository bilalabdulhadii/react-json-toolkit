## Overview

Json Toolkit is a single-page React application for working with JSON directly in the browser.

It focuses on the core developer workflow:

- paste or upload JSON
- format or minify it
- validate syntax safely
- inspect structured output in a tree view
- switch to raw output when needed
- copy or download the result

Everything runs locally in the browser. No JSON is uploaded, stored, or sent to a server.

---

## Features

- Developer-style split workspace with line-numbered input and output panes
- JSON formatting with readable indentation
- JSON minification for compact output
- Safe validation with clear syntax errors
- Tree viewer with collapsible nodes and inline value copy actions
- Raw output mode for exact formatted text
- File upload and drag-and-drop support
- Copy and download utilities
- Dark and light themes with persistent preference
- Mobile-first responsive layout
- Floating notifications that do not shift the toolbar layout

## Installation & Setup

1. Clone the repository.

```bash
git clone https://github.com/bilalabdulhadii/react-json-toolkit.git
cd react-json-toolkit
```

2. Install dependencies.

```bash
npm install
```

3. Start the development server.

```bash
npm start
```

The app will be available at `http://localhost:3000`.
